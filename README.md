# Atık sınıflandırma

## Nasıl kullanılıyor?

* Kullanıcı atığın fotoğrafını siteye yüklüyor site ise bu fotoğrafı sınıflayarak hangi atık kutusuna atılması gerektiğini söylüyor.

## Bu proje ne işe yarıyor?

* Bu proje atıkların doğru atık kutusuna atılarak çevre kirliliğinin önüne geçilmesine yardımcı oluyor.

## Hangi siteler kullanılacak 
* Google Teachable machine
* Python dili kullanılacak
* HTML/CSS
